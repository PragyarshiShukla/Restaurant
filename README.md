# Restaurant Website Landing Page

A simple static restaurant landing page built with HTML and CSS.

## Overview

This project showcases a restaurant website concept featuring:
- Hero section with a promotional offer
- Fast delivery, fresh ingredients, and discount highlights
- Menu preview gallery
- Footer with navigation and newsletter signup

## Files

- `index.html` - Main page structure and content
- `style.css` - Styling for layout, typography, and responsive design
- `images/` - Image assets used throughout the page

## Usage

1. Open `index.html` in a web browser.
2. Review or edit `style.css` to adjust the design.
3. Replace images in `images/` to customize the visual content.

## Notes

- This is a static front-end project and does not include JavaScript or backend functionality.
- Ideal for presenting a restaurant landing page prototype or portfolio piece.
